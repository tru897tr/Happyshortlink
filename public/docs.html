Tài liệu API

Toàn bộ dữ liệu được lưu trong bộ nhớ RAM của server — không sử dụng cơ sở dữ liệu ngoài. Mọi response đều theo định dạng JSON đồng nhất: { success, data } khi thành công, hoặc { success: false, error: { code, message } } khi có lỗi.

POST /api/links

Tạo một liên kết rút gọn mới từ URL gốc.

Request body
{ "url": "https://example.com/duong-dan-rat-dai", "customCode": "my-link", // không bắt buộc "ttlMinutes": 1440 // không bắt buộc — thời gian sống (phút) }
Response 201
{ "success": true, "data": { "code": "aZ3kP9q", "originalUrl": "https://example.com/duong-dan-rat-dai", "shortUrl": "https://your-app.onrender.com/link/aZ3kP9q", "createdAt": 1771600000000, "expiresAt": null } }
GET /api/links/:code

Lấy thông tin chi tiết một liên kết (không tăng lượt nhấp). Trang đếm ngược dùng endpoint này để biết link đích.

GET /api/links?limit=50&offset=0

Lấy danh sách các liên kết đã tạo, sắp xếp theo mới nhất.

DELETE /api/links/:code

Xoá một liên kết rút gọn khỏi hệ thống.

POST /api/posts

Đăng một bài viết mới gồm tiêu đề và nội dung đầy đủ.

Request body
{ "title": "Tiêu đề bài viết", "content": "Nội dung đầy đủ của bài viết...", "author": "Nguyễn Văn A", // không bắt buộc "tags": ["tin-tuc", "cong-nghe"], // không bắt buộc "coverImage": "https://..." // không bắt buộc }
Response 201
{ "success": true, "data": { "id": "k29sJ1mzQa", "slug": "tieu-de-bai-viet", "title": "Tiêu đề bài viết", "excerpt": "Nội dung đầy đủ của bài viết...", "url": "https://your-app.onrender.com/post/tieu-de-bai-viet" } }
GET /api/posts?limit=20&offset=0&tag=tin-tuc

API "ping" — lấy danh sách bài viết mới nhất. Mỗi item gồm tiêu đề và đoạn trích (excerpt), dùng để hiển thị trên trang chủ. Gọi định kỳ để theo dõi bài viết mới.

Tham sốKiểuMặc địnhMô tả
limitnumber20 (tối đa 100)Số bài viết mỗi trang
offsetnumber0Vị trí bắt đầu
tagstring—Lọc theo thẻ chủ đề
GET /api/posts/:idOrSlug

Lấy đầy đủ nội dung một bài viết (tiêu đề + toàn bộ nội dung), tăng lượt xem.

PUT /api/posts/:id

Cập nhật một phần hoặc toàn bộ nội dung bài viết.

DELETE /api/posts/:id

Xoá một bài viết khỏi hệ thống.
